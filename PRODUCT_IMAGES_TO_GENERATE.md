# Culture Bio Diamant — Prompts de génération d'images produits

## Format cible
- Dimensions : **800 × 800 px** (carré) ou **800 × 600 px** (paysage)
- Format de sortie : **WebP**
- Chemin de destination : `/public/products/{categorie}/{slug}.webp`
- Style global : fond sombre (noir profond ou charcoal #0a0a0f), éclairage dramatique, esthétique premium/luxury, net et épuré
- Interdiction : aucun texte dans l'image, aucun logo, aucune mention de THC ou de marque concurrente

## Style commun à toutes les images
> Fond noir mat profond. Éclairage spot dramatique d'en haut ou de côté. Reflets brillants sur le produit. Esthétique photographique macro/close-up professionnelle. Rendu 3D photoréaliste ou photographie studio haut de gamme. Pas de personne, pas de main humaine visible.

## Instructions d'utilisation

### Outils recommandés
- **Midjourney v6** — meilleur rendu réaliste pour produits
- **DALL-E 3** (via ChatGPT) — bon pour produits simples
- **Adobe Firefly** — conforme droits d'auteur

### Paramètres Midjourney recommandés
```
--ar 1:1 --style raw --v 6 --q 2
```
Ajouter en suffix à chaque prompt.

### Processus
1. Générer l'image avec le prompt
2. Sauvegarder en WebP, 800×800px, qualité 85%
3. Nommer exactement comme indiqué : `{slug}.webp`
4. Placer dans `/public/products/{categorie}/`
5. L'image sera automatiquement chargée par `ProductCard.tsx`

### Fallback automatique
En l'absence d'image `.webp`, `ProductCard.tsx` charge automatiquement `/products/{categorie}/fallback.svg`.
